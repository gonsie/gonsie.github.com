3 Setup

3.1 Things You'll Need

  • Resume
  • Text editor with Markdown Support
  • GitHub account
  • Package manager to install Jekyll (optional)

3.2 Text Editor with Markdown Support

  • Syntax highlighting
  • Export / preview support (may require additional software)

3.3 Text Editor with Markdown Support

  • Emacs (markdown-mode / MELPA)
  • Vim (vim-markdown)
  • SublimeText (packages: markdown)
  • Atom (packages: markdown)
  • VSCode (extensions: markdown)

3.4 Package Managers

  1. MacOS
    • Homebrew package manager
  2. Windows
    • Windows Subsystem for Linux
      • Suggested Linux: Ubunto Image
  3. Linux
    • apt-get for Ubuntu/Debian distros
    • yum for RedHat/CentOS distros

4 Activities

4.1 Activity 2: GitHub pages

  1. GitHub Pages
    • Publish website with GitHub Pages
    • Free hosting for project and user websites
      • User website in a repo with a special name: username.github.io
      • Project website in a branch with a special name: gh-pages (usually). This will appear at https://username.github.io/projectname
  2. Activity 2: Resume with GitHub pages
    1. Fork the gonsie/Cthulu-Resume: one-page resume repo
    2. Rename the repo to username.github.io
    3. Clone the repo to your computer
    4. Edit the index.html file with your own details
    5. Publish your changes
    6. View the changes online: https://username.github.io
